    GitHub Actions: Overview

    GitHub Actions: Overview

    A GitHub Action can perform a variety of tasks, including automating the build and deployment of a MarkBind site.

    • markbind-action is a published action that builds and deploys your MarkBind site.
    • markbind-reusable-workflows is a list of reusable workflows that help improve your CI/CD with a MarkBind site.

    Before any development, ensure you have a basic knowledge of GitHub Actions
